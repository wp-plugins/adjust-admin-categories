== Description ==
Installing this plugin allows you to adjust the behavior of the area below the posts screen category and custom taxonomy box. 

1. Stop the checkbox from moving
2. Change checkbox to radio button
3. Make parent categories unselectable 
4. Required to Category or Taxonomies


== Installation ==
1. Upload the entire `adjust-admin-categories` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The control panel of Adjust Admin Categories is in 'Settings > Adjust Categories'.
